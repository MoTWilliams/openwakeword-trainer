# OpenWakeWord Trainer

Application for recording sound clips from the Seeed Wio Terminal and organizing them into a dataset for training an OpenWakeWord wake word model.

This project is intended to make local wake-word dataset collection repeatable in the same environment where the wake word system will eventually be used. The Wio Terminal records audio from its intended wall-mounted location, while a mobile-friendly web interface allows recordings to be started, stopped, reviewed, and labeled wirelessly from different positions in the house. Heavier processing, storage, metadata tracking, and dataset organization are handled by a local server.

This project is currently a placeholder. It will eventually be built using prototypes from [Wio Terminal Prototypes and Labs](https://github.com/MoTWilliams/wio-terminal).
